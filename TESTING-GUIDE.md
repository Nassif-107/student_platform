Перед тестированием убедитесь что все работает:
```bash
docker compose -f docker-compose.dev.yml up -d --build
docker compose -f docker-compose.dev.yml exec backend pnpm seed
```

Открыть http://localhost:3000

## Сценарий 1: Регистрация и авторизация

### 1.1 Регистрация нового пользователя
1. Открыть http://localhost:3000
2. Должна открыться страница входа
3. Нажать "Зарегистрируйтесь"
4. Заполнить форму:
   - Имя: Новый
   - Фамилия: Пользователь
   - Email: newuser@test.ru
   - Пароль: testpassword123
   - Университет: КубГТУ
   - Факультет: Институт компьютерных систем
   - Направление: Прикладная информатика
   - Курс: 2
5. Нажать "Зарегистрироваться"
6. Проверить: перенаправило на главную страницу
7. Проверить: в правом верхнем углу отображается имя

### 1.2 Выход из системы
1. Нажать на аватар в правом верхнем углу
2. Нажать "Выйти"
3. Проверить: перенаправило на страницу входа

### 1.3 Вход существующим пользователем
1. На странице входа ввести:
   - Email: student3@university.ru
   - Пароль: password123
2. Нажать "Войти"
3. Проверить: главная страница с приветствием "Привет, Дмитрий!"

### 1.4 Вход с неправильным паролем
1. Ввести email: student3@university.ru
2. Ввести пароль: wrongpassword
3. Нажать "Войти"
4. Проверить: появилось сообщение об ошибке "Неверный email или пароль"

### 1.5 Вход с несуществующим email
1. Ввести email: nobody@test.ru
2. Ввести пароль: anything
3. Проверить: ошибка

## Сценарий 16: Навигация и UX

1. На любой странице нажать Ctrl+K (или Cmd+K на Mac)
2. Проверить: открылась палитра команд
3. Набрать "курсы"

8. Проверить: палитра закрылась

### 16.2 Хлебные крошки
1. Перейти на страницу деталей курса
2. Проверить: хлебные крошки "Главная > Курсы > Название курса"
3. Нажать "Курсы" в хлебных крошках
4. Проверить: вернулись на список курсов

### 16.3 Прокрутка наверх
1. Прокрутить длинную страницу вниз
2. Проверить: появилась кнопка "наверх" в правом нижнем углу
3. Нажать на неё
4. Проверить: страница прокрутилась наверх

### 16.4 Полоса прогресса прокрутки
1. На длинной странице прокрутить вниз

### 16.5 Пустые состояния

1. На странице курсов прокрутить вниз
